# Susan's Company — Nairobi E-Commerce Website

A professional e-commerce website for Susan's Company,
a Nairobi shop selling modest clothing and books.

## Live Website
[susan-s-company.vercel.app](https://susan-s-company.vercel.app)

## Features
- Dynamic product shop connected to Supabase database
- Customer contact form with WhatsApp integration
- Order tracking system
- Secure admin panel for shop management
- Mobile responsive design
- Built with HTML, CSS, JavaScript and Supabase

## Pages
- Homepage (index.html)
- Shop with filters (shop.html)
- Contact form (contact.html)
- Admin panel (admin-login.html)

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Database: Supabase (PostgreSQL)
- Hosting: Vercel
- Version Control: GitHub
